# 配置说明

## 全局配置

配置文件位于 `~/.simpleCode/settings.json`。

### Provider 配置项

| 配置项    | 说明                                              |
|-----------|---------------------------------------------------|
| `name`    | Provider 名称                                     |
| `baseUrl` | API 基础 URL                                      |
| `key`     | API 密钥                                          |
| `api`     | API 类型，对应 Registry 中注册的名称，如 `OpenAI` |

### Model 配置项

| 配置项     | 说明                                                 |
|------------|------------------------------------------------------|
| `provider` | 所属 Provider 名称                                   |
| `name`     | 模型名称                                             |
| `level`    | 模型等级，数字越小越强，0 为主模型，数字越大模型越弱 |

## 添加 Provider

在欢迎界面选择"添加 Provider"，填写：

1. **名称**: 自定义 Provider 名称
2. **Base URL**: API 端点地址
3. **Key**: API 密钥
4. **API**: API 类型，目前内置 `OpenAI`

## 模型选择

在欢迎界面，系统会从所有 Provider 获取可用模型列表。按数字键 `0-9` 可将模型添加到配置中，并设置其 level。

level 用于模型分级：

- `0`: 主模型，默认使用
- 数字越大，模型越弱，通常用于安全评估等轻量任务

## CLI 配置

CLI 配置保存在 `~/.simpleCode/cli.json`。

| 配置项     | 说明     | 默认值 |
|------------|----------|--------|
| `language` | 界面语言 | zh_cn  |
